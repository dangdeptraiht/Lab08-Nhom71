## 2. Tôi đã ra một quyết định kỹ thuật gì?

**Quyết định:** Dùng **rule-based keyword matching** trong `analyze_policy()` thay vì gọi LLM để phân tích policy.

**Các lựa chọn thay thế:**
- **LLM-based** (GPT-4o-mini classify): chính xác hơn, hiểu negation, nhưng thêm ~800ms/câu và thêm một API call vào worker đã có synthesis.
- **Rule-based** (cách tôi chọn): deterministic, không thêm latency, đủ coverage cho 5 exception types trong `contracts/worker_contracts.yaml`.

**Lý do chọn rule-based:**  
Sprint 2 cần unblock Sprint 3 và Sprint 4 sớm nhất. Rule-based giúp workers chạy được end-to-end ngay mà không phụ thuộc thêm API key hay model mới.

**Trade-off đã chấp nhận:**  
Rule-based không hiểu negation. Câu q12 — *"không phải Flash Sale"* — bị false positive:

```python
# workers/policy_tool.py  hàm analyze_policy(), dòng ~50
if "flash sale" in task_lower or "flash sale" in context_text:
    exceptions_found.append({"type": "flash_sale_exception", ...})
```

`"flash sale" in task_lower` trả về `True` cho câu chứa "không phải Flash Sale" → `policy_applies = False` sai. Trace `run_20260414_163641.json` ghi `exceptions: ["flash_sale_exception"]` dù task nói rõ không phải Flash Sale. Đây là hạn chế đã biết, ghi nhận để cải tiến.

## 3. Tôi đã sửa một lỗi gì?

**Lỗi:** Workers không load `.env` khi chạy standalone → ChromaDB query fail với 0 chunks.

**Symptom:**  
Chạy `python workers/retrieval.py` ra kết quả `Retrieved: 0 chunks` cho mọi query, kèm warning:
```
⚠️  ChromaDB query failed: Collection expecting embedding with dimension of 1536, got 384
```

**Root cause:**  
`retrieval.py` không có `load_dotenv()`. Khi chạy standalone, `OPENAI_API_KEY` không được load từ `.env` → `os.getenv("OPENAI_API_KEY", "")` trả về chuỗi rỗng → `_get_embedding_fn()` fallback sang Sentence Transformers (384-dim). Nhưng collection `day09_docs` đã index bằng OpenAI embeddings (1536-dim) → dimension mismatch → query fail hoàn toàn.

**Cách sửa:**  
Thêm đoạn load dotenv vào đầu cả ba worker files:

```python
# Thêm vào workers/retrieval.py, workers/policy_tool.py, workers/synthesis.py
try:
    from dotenv import load_dotenv
    load_dotenv()
except ImportError:
    pass
```

**Bằng chứng trước/sau:**

*Trước khi sửa:*
```
▶ Query: SLA ticket P1 là bao lâu?
  Retrieved: 0 chunks
  Sources: []
```

*Sau khi sửa:*
```
▶ Query: SLA ticket P1 là bao lâu?
  Retrieved: 3 chunks
    [0.683] sla-p1-2026.pdf: Tai lieu: SLA TICKET - QUY ĐỊNH XỬ LÝ SỰ CỐ ...
  Sources: ['sla-p1-2026.pdf']
```
